# Civic Sense 🚀

*A Smart Community Issue Reporting Platform*

## 📌 Overview

Civic Sense is a community-driven platform that empowers citizens to report local civic issues such as potholes, drainage problems, garbage overflow, and streetlight failures.

Users can submit complaints through **photo uploads, live streaming, or file uploads**, which are then forwarded to the **concerned authorities** for timely action.

---

## ✨ Key Features

### 📸 Easy Issue Reporting

* Report issues via photo, live stream, or file upload
* Location-based tagging for accurate reporting

### 🗺️ Real-time Map Updates

* All reported issues are displayed on a map
* Continuous status updates for each issue
* Helps travelers avoid problematic areas

### 🏆 Reward System

* Users earn **points** based on the number of reports submitted
* Points can be redeemed for exciting rewards

### ✅ Smart Verification System

* If the same issue is reported by **2 or more users**,
  it is automatically marked as **verified**

### 🏛️ Authority Action Tracking

* Authorities can update the actions taken on each issue
* Users can track progress in real time

### ⏰ Reminder & Accountability System

* If no action is taken within **1 week**,
  an automatic reminder is sent to the authority
* If still unresolved, the authority is moved to a
  **flagged/block list** for accountability

---

## 🎯 Objective

* Bridge the gap between citizens and authorities
* Ensure faster issue resolution
* Promote transparency and accountability in governance
